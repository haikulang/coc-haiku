# coc-haiku

Nvim client for Haiku Language Server

## Install

`:CocInstall coc-haiku`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-haiku-keymap)`

## Lists

`:CocList demo_list`
